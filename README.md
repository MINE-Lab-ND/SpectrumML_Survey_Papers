## 🌟 Introduction
This repository collects papers at the intersection of **machine learning** and **spectroscopic analysis** (including but not limited to MS, NMR, IR, Raman, UV-Vis, etc.).  
All entries are organized by **Task** (e.g., structure elucidation, spectra prediction, functional group prediction, etc.).  
If a paper addresses multiple tasks, it will appear under multiple headings.  

We welcome contributions and suggestions—please feel free to open an issue or submit a pull request!
